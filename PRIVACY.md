# Privacy Policy

Kondreh is a private macOS camera preview utility.

## Data Accessed

Kondreh accesses the selected camera only after the user grants macOS Camera permission.

## Data Processed

Camera frames are processed locally by AVFoundation for display in an `AVCaptureVideoPreviewLayer`.

## Data Stored

Kondreh stores local preferences in `UserDefaults`, including selected camera identifier, mirror preference, aspect ratio, preview size, appearance, shortcut, onboarding completion, and trial metadata.

## Data Not Collected

Kondreh does not:

- Record video
- Capture still images
- Save camera frames to disk
- Upload camera frames
- Transmit camera frames to another process
- Request microphone access
- Use analytics
- Use crash reporting
- Use advertising SDKs
- Use trackers
- Require an account
- Sync camera data to the cloud

## Network Use

The current implementation has no network entitlement and no backend. Update and production license providers are intentionally not connected in this development build.

## Permissions

Kondreh requests Camera permission with this macOS usage description:

> Kondreh needs camera access to show you a private preview before video calls. Nothing is recorded or uploaded.
